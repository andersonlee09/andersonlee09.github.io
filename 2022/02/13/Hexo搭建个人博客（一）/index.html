Hexo 搭建个人博客

Hexo 搭建个人博客

0.说明

本项目基于Windows10利用Hexo部署个人博客并且投放至GitHub

1.下载node.js

顺带下载npm

直接在官网下载，本文不作说明

2.查看环境

node -v

npm -v

如果没有异常则下载成功

3.切换镜像

npm install -g cnpm –registry=https://registry.npm.taobao.org

4.下载hexo

cnpm install -g hexo-cli

hexo -v查看版本

5.初始化博客

hexo init

初始化结果：

6.启动博客

hexo s

初始化结果

7.写博客

hexo n “My first blog”

对该目录下markdown进行编辑

编辑结束后

hexo clean

hexo g

hexo s

结果：

8.投放至GitHub

8.1 创建GitHub项目

命名必须符合规范：name.github.io

8.2下载git帮助

npm install hexo-deployer-git –save

8.3设置_config.yml

8.4 修改Deployment

修改前：

修改后：

hexo g -d

8.4成功部署

至此所有部署已经完成
